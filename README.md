# HackASM
### Assembler for the [nand2tetris](https://www.nand2tetris.org/) Hack platform written in Rust.

## Usage

```$ hackasm <assembly file>```

The file must have a `.asm` extension, and the output will be the
file name with a `.hack` extension.
